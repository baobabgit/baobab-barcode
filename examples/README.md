# Exemples

Le point d’entrée recommandé est la **façade publique** documentée dans le [README principal](../README.md) (`generate`, `validate_payload`, `decode_from_bytes`, `decode_from_file`).

Le fichier `basic_usage.py` propose un script minimal exécutable depuis la racine du dépôt après `pip install -e .` :

```bash
python examples/basic_usage.py
```
